Introduction
Bike-sharing systems are becoming increasingly prevalent in urban environments. They provide a low-cost, environmentally-friendly transportation alternative for cities. New York City launched the largest bike-sharing system in North America, Citibike. The system has been a success with ridership approaching 62,000 trips per day (October 2017).
Data
The data comes from citibike system data website, I chose two months data: June 2015 and January 2016. The data includes: Trip Duration( seconds) , Start Time and Date, Stop Time and Date, Start Station Name
  • End Station Name
  • Station ID
  • Station Lat/Long
  • Bike ID
  • User Type (Customer = 24-hour pass or 3-day pass user; Subscriber = Annual Member)
  • Gender (Zero=unknown; 1=male; 2=female)
  • Year of Birth
